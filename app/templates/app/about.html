What is The Blog?

The Blog is a blogging CRUD (Create, Retrieve, Update, Delete) web application built using the django framework.
The web application enables a user to:
1. Post articles.
2. View all or articles belonging to him/her.
3. Update their articles as well as their user profile.
4. Delete articles.
On top of all these, the application also enables a user to comment on other users' articles.
